Variant rationales (full descriptions)

KeySectionRationale
bid_dealer_open_2 bid When all three opponents pass, the dealer currently bids 1 (downtown signal). This variant opens with bid 2 (uptown signal) instead. Tests the hypothesis that uptown is a safer default direction when the dealer has no info about partner strength.
bid_no_bid5_via_length bid The early-bidder rule "max_suit_count() >= 7 → bid 5" commits to a 5 contract on length alone without checking for honors. Tests whether removing it helps (too optimistic about bare length) or hurts (missed legit opportunities).
bid_seat3_bid_only_3 bid The hot-seat rule "always bid 4 if current is below 4" forces a 4 contract even on weak hands. This variant lets seat 3 bid 3 instead when current bid < 3, keeping the commitment lower until the dealer can respond.
lead_pull_trump_min2 leading The baseline pulls trump when enemies still have any trump at all — even a single outstanding card. This variant requires at least 2 outstanding before committing a high trump to pull. Saves trump for later trump tricks.
lead_pull_trump_min3 leading More conservative variant of the 2+ rule — require 3 outstanding trump.
follow_overtake_1_threat following Currently when partner is winning the trick, we only duck (play weakest) if there are ZERO outstanding threats. This variant ducks even with 1 threat — i.e. we're more willing to let partner's winner ride. Tests whether the overtake logic was too trigger-happy.
void_no_signal_first void Baseline: when I'm void on lead suit, partner isn't winning, and I have trump, I trump. Otherwise if I haven't signaled yet, I signal with a non-trump discard. This variant removes the signal-first pathway entirely — either trump or play weakest. Tests whether the signal is worth the trump we're NOT using when partner wins cheap.
void_trump_strongest void When I trump into a trick, baseline plays the weakest trump that will win. This variant plays the strongest trump instead. The gamble: using a big trump guarantees the trick (even if a bigger trump is outstanding) and drains my small trump as "currency" for later tricks.
discard_keep_2 discard When partner signals, baseline keeps 1 card per suit in the discard keep-set. This variant keeps 2 per suit — more conservative suit retention. Tests whether giving up depth for void creation is the right tradeoff.
